# Seleted text translator for Min browser

Adds translating selected text by LibreTranslate
<img width="777" height="429" alt="image" src="https://github.com/user-attachments/assets/69e3955e-f4c2-4d31-aad3-ad40f3321b95" />

## Configuration & Installation

* Enable userscripts in Min and create a `userscripts` folder following [these instructions](https://github.com/minbrowser/min/wiki/userscripts), and save `translate-selected.js` inside it.
* Provide your [libretranslate](https://docs.libretranslate.com/) api url and target translation language inside userscript
* You can also configure styles for floating popup menu there
* Restart Min.

## Usage
* You can double click the word or selecting text with alt/option key by default
